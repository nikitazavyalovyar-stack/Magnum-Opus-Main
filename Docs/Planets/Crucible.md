# 🔥 Планета "Горнило" (The Crucible)

## Основная информация

| Параметр | Значение |
|----------|----------|
| **Тип** | PvP-планета высокого риска |
| **Размер** | 1800 x 1800 метров |
| **PvP** | Полный (везде, без исключений) |
| **Гравитация** | 0.85 (ниже земной) |

| **Цикл дня/ночи** | 20 минут реального времени |

---

## 📖 Концепция

**Горнило** — это планета для опытных игроков, где риск равен награде. Здесь находится основной фарм валюты, редчайшие ресурсы и самые опасные PvE/PvP зоны.

**Философия:** "Выживает сильнейший. Но даже сильнейший может потерять всё."

---

## 🗺️ Биомы

### 🌋 Вулканический биом (35% поверхности)
**Расположение:** Центральная часть  
**Особенности:**
- Активные вулканы с лавовыми потоками
- Серные испарения (урон здоровью без защиты)
- Редчайшие минералы (титан, платина, иридий)
- Высокий риск PvP (все приходят за ресурсами)

**Ресурсы:**
- Титановая руда (для тяжёлой брони)
- Платиновая руда (для электроники)
- Иридий (для квантовых двигателей)
- Сера (для взрывчатки)
- Обсидиан (для острого оружия)

**Опасности:**
- Лава (мгновенная смерть)
- Серные газы (урон 10 HP/сек без скафандра)
- Вулканические бомбы (случайные взрывы)

### 🏔️ Кристаллический биом (25% поверхности)
**Расположение:** Северная часть  
**Особенности:**
- Гигантские кристаллические formations
- Пещеры с редкими артефактами
- Аномальная гравитация (в некоторых зонах)
- Элитные NPC-стражи (высокий уровень)

**Ресурсы:**
- Энергетические кристаллы (для щитов)
- Кварц (для оптики и лазеров)
- Алмазы (для режущих инструментов)
- Редкоземельные элементы (для продвинутой электроники)

**Опасности:**
- Кристаллические осколки (урон при взрыве)
- Гравитационные аномалии (засасывают игрока)
- Элитные NPC (уровень 50+)

### 🏜️ Пустошь (25% поверхности)
**Расположение:** Южная часть  
**Особенности:**
- Радиоактивные зоны (нужна защита)
- Заброшенные военные базы (лут)
- Мутанты (PvE враги)
- Зоны с артефактами древней цивилизации

**Ресурсы:**
- Уран (для ядерных реакторов)
- Плутоний (для оружия массового поражения)
- Военные чертежи (редкие рецепты)
- Древние артефакты (уникальные предметы)

**Опасности:**
- Радиация (урон 5 HP/сек без защиты, накапливается)
- Мутанты (агрессивные NPC)
- Ловушки на военных базах

### 🌑 Тёмная зона (15% поверхности)
**Расположение:** Восточная часть (вечная ночь)  
**Особенности:**
- Постоянная темнота (нужны фонари)
- Самые редкие ресурсы во всей игре
- Элитные PvP-игроки охотятся здесь
- Боссы мирового уровня

**Ресурсы:**
- Тёмная материя (для квантовых технологий)
- Антиматерия (для супероружия)
- Звёздная пыль (для уникальных улучшений)
- Чертежи легендарного оружия

**Опасности:**
- Темнота (видимость 10 метров без фонаря)
- Элитные PvP-игроки (уровень 60+)
- Мировые боссы (уровень 100, рейд на 10+ игроков)

## 📍 Ключевые локации

### Зона приземления (0, 0, 0)
**Тип:** Нейтральная зона (PvP запрещён 500м)  
**Описание:**
- Посадочные площадки для кораблей
- Торговая станция (нейтральная территория)
- Медицинский центр (лечение, страховка)
- Информационный центр (карта, квесты)

**Правила:**
- PvP запрещён в радиусе 500м
- Можно купить/продать ресурсы
- Можно застраховать снаряжение
- Можно нанять охрану (NPC-эскорт)

### Вулканическая крепость
**Расположение:** (300, 0, 400)  
**Тип:** Рейдовая локация (PvE + PvP)  
**Описание:**
- Древняя крепость в жерле вулкана
- Босс "Хранитель Горнила" (уровень 80)
- Легендарный лут (оружие, броня)
- 10-20 игроков одновременно

**Механика:**
- Рейд на 5-10 игроков
- Босс имеет 3 фазы
- PvP разрешён во время рейда (можно атаковать других)
- Лут распределяется по системе DKP (очки вклада)

### Заброшенная военная база "Омега"
**Расположение:** (-500, 0, -300)  
**Тип:** Исследовательская локация  
**Описание:**
- Подземный бункер с 5 уровнями
- Ловушки и головоломки
- Элитные NPC-охранники
- Чертежи экспериментального оружия

**Механика:**
- Одиночное прохождение или группа до 4 игроков
- Головоломки (коды, рычаги, лазерные сетки)
- Скрытые комнаты с секретным лутом
- Финальный босс "Командир базы" (уровень 60)

### Арена "Колизей"
**Расположение:** (0, 0, -500)  
**Тип:** PvP-арена  
**Описание:**
- Круглая арена для дуэлей и турниров
- Зрительские трибуны (другие игроки могут смотреть)
- Ставки на победителя (койны)
- Еженедельные турниры с призами

**Механика:**
- Дуэли 1v1
- Командные бои 3v3, 5v5
- Режим "Король горы"
- Ставки: победитель получает 2x ставку

## ⚔️ PvP-система

### Полный PvP
**Правила:**
- Можно атаковать любого игрока вне нейтральных зон
- При убийстве — весь лут жертвы выпадает (кроме застрахованного)
- Нет системы "преступник" (все равны)
- Можно формировать альянсы и гильдии

### Система репутации
**Типы репутации:**
- **Нейтральная** (по умолчанию)
- **Охотник** (убил 10+ игроков за неделю) — бонус к урону
- **Жертва** (убит 10+ раз за неделю) — бонус к защите
- **Легенда** (убил 100+ игроков) — уникальный титул

### Страховка
**Важно:** На Горниле страховка работает иначе:
- Обычная страховка: возврат 50% стоимости
- Премиум-страховка: возврат 80% стоимости (дороже)
- Если игрок убит в PvP — страховка НЕ возвращается (если убийца забрал лут)

## 💰 Экономика Горнила

### Редкие ресурсы (только здесь)
| Ресурс | Цена (койны) | Использование |
|--------|--------------|---------------|
| Титан | 500/кг | Тяжёлая броня, корабли |
| Платина | 1200/кг | Электроника, щиты |
| Иридий | 3000/кг | Квантовые двигатели |
| Уран | 2000/кг | Ядерные реакторы |
| Тёмная материя | 10000/кг | Квантовые технологии |
| Антиматерия | 50000/кг | Супероружие |

### Фарм валюты
**Способы заработать:**
1. **Добыча ресурсов** — 500-5000 койнов/час
2. **Убийство игроков** — лут жертвы (может быть 10000+ койнов)
3. **Рейды** — легендарный лут (50000+ койнов)
4. **Квесты** — 2000-10000 койнов за задание
5. **Торговля** — покупка дёшево на Колыбели, продажа дорого на Горниле

### Риски
- Потеря снаряжения при смерти (если не застраховано)
- Потеря ресурсов в инвентаре
- Потеря корабля (если уничтожен и не застрахован)

## 🎮 Геймплей на планете

### Для опытных игроков (30-50 уровень)
1. Прибытие на Горнило (корабль или телепорт)
2. Покупка страховки и снаряжения
3. Фарм ресурсов в вулканическом биоме
4. PvP-стычки с другими игроками
5. Продажа ресурсов на торговой станции

### Для элитных игроков (50+ уровень)
1. Рейды на вулканическую крепость
2. Исследование тёмной зоны
3. Участие в турнирах на арене
4. Охота на других элитных игроков
5. Контроль территорий (гильдии)

### Для гильдий
1. Захват и удержание ресурсных точек
2. Войны гильдий (контроль территорий)
3. Рейды на мировых боссов
4. Контроль торговой станции (налоги)

## 🛠️ Технические детали

**Сцена Unity:** `Assets/Scenes/Planets/Crucible.unity`  
**Terrain:** 1800x1800, высота до 800м (вулканы)  
**Освещение:** Directional Light + Volumetric Fog (серные испарения)  
**Оптимизация:**
- LOD для кристаллов и скал
- Occlusion Culling для пещер
- Particle System для лавы и газов
- Dynamic Weather (пепельные бури)

**Специальные эффекты:**
- Heat Distortion (искажение от жары)
- Radiation Overlay (зелёный туман в радиоактивных зонах)
- Darkness System (ограничение видимости в тёмной зоне)
