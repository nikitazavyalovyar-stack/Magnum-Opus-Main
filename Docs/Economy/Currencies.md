# 💰 Экономика: Валюты и баланс (Magnum Opus Economy System)

## 📖 Описание

**Экономическая система** — это сложная сеть взаимодействия валют, баланса и монетизации. Вдохновлена EVE Online, Star Citizen и Tarkov, но с уникальными особенностями для космической MMO.

**Философия:** "Деньги делают игру. Но не переборщи с балансом."

---

## ⚙️ Основные принципы

### 1. Двойная валютная система
В игре две валюты: **Койны (Coins)** и **Золото (Gold)**.

| Валюта | Тип | Как заработать | Как потратить | Особенности |
|--------|-----|----------------|---------------|-------------|
| **Койны (Coins)** | Игровая валюта | - Добыча ресурсов<br>- Продажа на аукционе<br>- Квесты | - Покупка чертежей<br>- Улучшение базы<br>- Строительство | - Не выводится в реальные деньги<br>- Обычная валюта (бесплатно) |
| **Золото (Gold)** | Реальная валюта | - Подписка (500₽/мес)<br>- Продажа через биржу | - Покупка редких предметов<br>- Продажа через биржу | - Выводится в реальные деньги<br>- Требует подписки |

### 2. Баланс валют
**Койны → Золото:** Можно конвертировать Койны в Золото через биржу, но с комиссией (10%).

**Золото → Койны:** Нельзя конвертировать обратно (только через биржу с комиссией 15%).

**Пример:**
- 1000 Койнов = 900 Золота (после комиссии)
- 1000 Золота = 850 Койнов (после комиссии)

### 3. Система комиссий
| Операция | Комиссия | Ограничение |
|----------|----------|-------------|
| Конвертация Койны → Золото | 10% | Минимум 100 Койнов |
| Конвертация Золото → Койны | 15% | Минимум 50 Золота |
| Покупка через биржу | 5% | Минимум 100 Койнов |
| Продажа через биржу | 3% | Минимум 100 Койнов |

**Цель:** Стимулировать покупку через биржу, а не конвертацию.

## 📊 Валюты и их источники

### 1. Койны (Coins)
**Как заработать:**
- **Добыча ресурсов** (10-1000/час)
  - Лес: 50/час
  - Горы: 200/час
  - Пустыня: 300/час
  - Океан: 100/час
- **Продажа на аукционе** (50-5000/час)
- **Квесты** (200-5000/час)
- **Рейды** (1000-10000/час)

**Как потратить:**
- **Покупка чертежей** (50-1000)
- **Строительство баз** (300-5000)
- **Крафт предметов** (100-3000)
- **Улучшение базы** (200-10000)
- **Система страхования** (50-2000)

### 2. Золото (Gold)
**Как заработать:**
- **Подписка** (500₽/мес = 5000 Золота)
- **Продажа через биржу** (100-5000/час)
  - Редкие предметы: 5000-10000
  - Техника: 1000-5000
  - Броня: 500-2000
- **Торговля с NPC** (200-3000/час)

**Как потратить:**
- **Покупка редких предметов** (100-5000)
- **Продажа через биржу** (100-5000)
- **Улучшение арены** (500-2000)
- **Приватные серверы** (1000-5000)
- **Премиум аккаунт** (1000/час)

## 💼 Система биржи и аукциона

### 1. Аукцион (Auction)
**Правила:**
- Можно выставить любой предмет (кроме уникальных)
- Минимальная ставка = 100 Койнов
- Тайм-аут: 24 часа
- Победитель платит комиссию 5%

**Пример:**
- Выставлено: Квантовый двигатель
- Минимальная ставка: 10000 Койнов
- Победитель: 12000 Койнов
- Итог: 12000 - 600 (комиссия) = 11400 Койнов

### 2. Биржа (Exchange)
**Правила:**
- Только для Золота
- Покупка/продажа через биржу
- Комиссия: 3% за продажу, 5% за покупку
- Минимум 100 Золота

**Пример:**
- Продано: Квантовый двигатель за 1200 Золота
- Итог: 1200 - 36 (комиссия) = 1164 Золота
- Куплено: Квантовый двигатель за 1200 Золота
- Итог: 1200 + 60 (комиссия) = 1260 Золота

## 🔄 Система экономики (Economic Flow)

**Как заработать Койны:**
1. Добыча ресурсов → Койны
2. Продажа на аукционе → Койны
3. Квесты → Койны
4. Рейды → Койны

**Как превратить Койны в Золото:**
1. Продажа через биржу → Золото (с комиссией)
2. Конвертация → Золото (с комиссией)

**Как заработать Золото:**
1. Подписка → Золото
2. Продажа через биржу → Золото

**Как потратить Золото:**
1. Покупка редких предметов → Золото
2. Улучшение арены → Золото
3. Приватные серверы → Золото
