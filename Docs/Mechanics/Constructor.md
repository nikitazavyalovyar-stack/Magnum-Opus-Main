# 🔧 Механика: Конструктор техники (Magnum Opus Vehicle Constructor)

## 📖 Описание

**Конструктор техники** — это модульная система сборки транспортных средств, космических кораблей и боевых машин из компонентов. Игроки могут создавать уникальную технику, комбинируя тысячи деталей с учётом физики, энергетики и баланса.

**Вдохновение:** Kerbal Space Program + Star Citizen + From the Depths + Besiege + Stormworks

**Философия:** "Если ты можешь это представить — ты можешь это построить. Но заставь ли ты это работать?"

---

## ⚙️ Основные принципы

### 1. Модульная архитектура
Каждая техника состоит из **компонентов** (Components), которые соединяются через **сокеты** (Sockets) и **узлы крепления** (Attachment Points).

**Иерархия:**

Техника (Vehicle)
├── Корпус (Hull/Frame)
│ ├── Кабина (Cockpit)
│ ├── Двигательный отсек (Engine Bay)
│ └── Грузовой отсек (Cargo Bay)
├── Двигатели (Engines)
├── Системы (Systems)
│ ├── Энергетика (Power)
│ ├── Охлаждение (Cooling)
│ └── Жизнеобеспечение (Life Support)
├── Вооружение (Weapons)
└── Защита (Defense)


### 2. Система сокетов (Socket System)
Каждый компонент имеет **входные** и **выходные** сокеты определённых типов:

**Типы сокетов:**
- **Structural** (Структурный) — для крепления к корпусу
- **Power** (Энергетический) — для подключения к энергосети
- **Data** (Информационный) — для управления и сенсоров
- **Fluid** (Жидкостный) — для топлива и хладагента
- **Mount** (Крепёжный) — для установки оружия/оборудования

**Размеры сокетов:**
- **Small** (S) — лёгкие компоненты
- **Medium** (M) — средние компоненты
- **Large** (L) — тяжёлые компоненты
- **Capital** (C) — компоненты для capital ships

**Совместимость:**
- S → S, M, L (маленький можно вставить в большой через адаптер)
- M → M, L
- L → L
- C → C (только для capital ships)

### 3. Физика и реализм
**Параметры физики:**
- **Масса** (Mass) — влияет на ускорение и инерцию
- **Центр масс** (Center of Mass) — влияет на устойчивость
- **Момент инерции** (Moment of Inertia) — влияет на вращение
- **Аэродинамика** (Aerodynamics) — сопротивление воздуха
- **Тяга** (Thrust) — сила двигателей

**Реалистичные ограничения:**
- Если центр масс слишком высоко — техника переворачивается
- Если тяга недостаточна — техника не взлетает
- Если аэродинамика плохая — техника разваливается на скорости

## 🚗 Типы техники

### Наземная техника (Ground Vehicles)

#### Лёгкая (Light)
**Масса:** до 2 тонн  
**Примеры:**
- Багги (Buggy)
- Мотоцикл (Motorcycle)
- Квадроцикл (ATV)
- Разведывательный вездеход (Scout Rover)

**Особенности:**
- Высокая скорость (150-250 км/ч)
- Низкая броня
- 1-2 места
- Малый расход топлива

#### Средняя (Medium)
**Масса:** 2-10 тонн  
**Примеры:**
- Внедорожник (SUV)
- Грузовик (Truck)
- Бронетранспортёр (APC)
- Артиллерийская установка (Artillery)

**Особенности:**
- Средняя скорость (80-150 км/ч)
- Средняя броня
- 2-6 мест
- Можно установить тяжёлое оружие

#### Тяжёлая (Heavy)
**Масса:** 10-100 тонн  
**Примеры:**
- Танк (Tank)
- Самоходная установка (SPG)
- Тяжёлый грузовик (Heavy Truck)
- Мобильная база (Mobile Base)

**Особенности:**
- Низкая скорость (30-80 км/ч)
- Тяжёлая броня
- 2-10 мест
- Capital-grade оружие

### Воздушная техника (Aerial Vehicles)

#### Атмосферная (Atmospheric)
**Типы:**
- **Самолёты** (Airplanes) — высокая скорость, нужны ВПП
- **Вертолёты** (Helicopters) — вертикальный взлёт, низкая скорость
- **Дроны** (Drones) — маленькие, дистанционное управление
- **Дирижабли** (Airships) — медленные, грузоподъёмные

**Параметры:**
- Максимальная высота: 10,000 метров
- Скорость: 200-1500 км/ч
- Нужны крылья/роторы для подъёмной силы

#### Космическая (Spacecraft)
**Классы:**
- **Fighter** (Истребитель) — 1-2 места, высокая маневренность
- **Corvette** (Корвет) — 2-5 мест, среднее вооружение
- **Frigate** (Фрегат) — 5-15 мест, тяжёлое вооружение
- **Destroyer** (Эсминец) — 15-50 мест, capital ship
- **Cruiser** (Крейсер) — 50-200 мест, capital ship
- **Capital Ship** (Капитальный корабль) — 200+ мест, линкор/авианосец

**Особенности:**
- Нет аэродинамики в вакууме (только тяга)
- Нужны двигатели для маневрирования (RCS)
- Квантовые двигатели для межпланетных перелётов

### Водная техника (Naval Vessels)

**Типы:**
- **Лодки** (Boats) — маленькие, быстрые
- **Катера** (Speedboats) — средние, вооружённые
- **Подводные лодки** (Submarines) — скрытность, торпеды
- **Корабли** (Ships) — большие, тяжёлое вооружение
- **Авианосцы** (Aircraft Carriers) — мобильная база

**Особенности:**
- Плавучесть (Buoyancy) — должна быть положительной
- Сопротивление воды (Drag) — выше чем воздух
- Глубина погружения (для подводных лодок)

## 🔌 Компоненты (Components)

### Двигатели (Engines)

#### Химические (Chemical)
**Принцип:** Сжигание топлива для создания тяги

| Тип | Тяга | Масса | Топливо | Эффективность | Пример |
|-----|------|-------|---------|---------------|--------|
| **Бензиновый** | 100-500 kN | 50-200 кг | Бензин | Средняя | V6, V8, V12 |
| **Дизельный** | 200-1000 kN | 100-500 кг | Дизель | Высокая | Для грузовиков, танков |
| **Реактивный** | 50-200 kN | 200-800 кг | Авиатопливо | Средняя | Турбореактивный |
| **Ракетный** | 500-5000 kN | 500-2000 кг | Ракетное топливо | Низкая | Для космоса |

**Параметры:**
- **Тяга** (Thrust) — сила в килоньютонах (kN)
- **Удельный импульс** (Specific Impulse, Isp) — эффективность (секунды)
- **Расход топлива** (Fuel Consumption) — кг/сек
- **Время работы** (Burn Time) — секунды до перегрева

#### Электрические (Electric)
**Принцип:** Электрический мотор + батарея/генератор

| Тип | Тяга | Масса | Энергия | Эффективность | Пример |
|-----|------|-------|---------|---------------|--------|
| **Электромотор** | 20-200 kN | 30-150 кг | Электричество | Высокая | Для наземной техники |
| **Ионный** | 0.1-5 kN | 50-200 кг | Электричество | Очень высокая | Для космоса (медленный разгон) |
| **Плазменный** | 10-100 kN | 100-500 кг | Электричество | Высокая | Для космоса |

**Особенности:**
- Требуют энергетической системы
- Не выделяют выхлопов
- Низкий уровень шума

#### Квантовые (Quantum)
**Принцип:** Использование квантовой энергии для гиперпространственных прыжков

| Тип | Тяга | Масса | Энергия | Эффективность | Пример |
|-----|------|-------|---------|---------------|--------|
| **Квантовый двигатель** | 1000-10000 kN | 500-5000 кг | Квантовая энергия | Очень высокая | Для космических кораблей |
| **Гиперпространственный двигатель** | 5000-50000 kN | 2000-20000 кг | Квантовая энергия | Средняя | Для межпланетных перелётов |

**Особенности:**
- Требуют квантовой энергетики
- Позволяют совершать гиперпространственные прыжки
- Высокая стоимость и сложность в обслуживании

## 🔫 Вооружение (Weapons)

### Кинетическое (Kinetic)
Использует физические снаряды. Высокий урон по броне, есть баллистика.

| Тип | Скорострельность | Урон | Дальность | Особенности |
|-----|------------------|------|-----------|-------------|
| **Репетитор (Repeater)** | Очень высокая | Низкий | Средняя | Перегрев ствола, большой расход патронов |
| **Автопушка (Autocannon)** | Средняя | Средний | Высокая | Разрывные снаряды, отдача влияет на технику |
| **Рельсотрон (Railgun)** | Очень низкая | Критический | Сверхвысокая | Требует колоссальной энергии, пробивает насквозь |

### Энергетическое (Energy)
Использует лазеры и плазму. Мгновенное попадание, нет баллистики.

| Тип | Режим работы | Урон | Особенности |
|-----|--------------|------|-------------|
| **Лазер (Laser Beam)** | Непрерывный луч | Средний/сек | Плавит броню, нагревает цель, требует постоянного удержания |
| **Импульсный лазер (Pulse Laser)** | Залпы | Высокий | Мгновенный урон, меньше греется, но ниже DPS |
| **Плазменная пушка (Plasma)** | Заряд -> Выстрел | Очень высокий | Медленные сгустки плазмы, взрываются при контакте, наносят AoE урон |

### Ракетное и Взрывное (Explosive)
Самонаводящиеся или баллистические ракеты.

| Тип | Наведение | Урон | Особенности |
|-----|-----------|------|-------------|
| **Ракета (Missile)** | Тепловое/Радарное | Высокий | Можно сбить системами ПВО (CIWS) |
| **Торпеда** | Акустическое/Проводное | Критический | Только для подводной техники |
| **Мина/Глубинная бомба** | Контактная/По глубине | Высокий | Зонный контроль |

### Специальное и Электронное (Special / EW)
| Тип | Эффект | Применение |
|-----|--------|------------|
| **ЭМИ (EMP)** | Отключает системы врага на 5-15 сек | Обездвижить технику, отключить щиты |
| **Трейсер (Tractor Beam)** | Притягивает/отталкивает цели | Захват грузов, спасение, абордаж |
| **Сканер/Радар** | Подсветка целей на карте | Разведка, наведение ракет |

## 🛡️ Системы и Защита

### Энергетика (Power)
- **Реактор (Reactor):** Генерирует энергию. Чем мощнее, тем больше греется.
- **Батареи (Batteries):** Буфер энергии для пиковых нагрузок.
- **Топливные баки:** Хранят химическое топливо. Взрываются при пробитии.

### Охлаждение (Cooling)
- **Радиаторы:** Пассивное охлаждение.
- **Жидкостное охлаждение:** Активное, требует насосов и жидкости.
- **Сброс тепла (Heat Vent):** Мгновенный сброс перегрева, но создаёт тепловую сигнатуру.

### Защита (Defense)
- **Броня (Armor):** Физическое поглощение урона. Разные материалы (Сталь, Титан, Композит).
- **Энергощиты (Shields):** Поглощают энергетический урон, требуют много энергии.
- **Активная защита (APS):** Автоматически сбивает подлетающие ракеты.
- **ECM (Electronic Countermeasures):** Глушит наведение вражеских ракет и радары.

## 🎮 Геймплей сборки

### 1. Верстак (Workbench)
- Игрок подходит к верстаку нужного класса (Лёгкий, Тяжёлый, Космический).
- Открывается интерфейс конструктора.

### 2. Режим редактора
- Камера от третьего лица.
- Сетка (Grid) для точного размещения.
- Превью всех компонентов.

### 3. Проверка (Validation)
- Игра подсвечивает красным ошибки (нет связи с кабиной, не хватает энергии).
- Показывает статистику техники (масса, ускорение, тяга).

### 4. Тест-драйв
- Запуск симуляции прямо в редакторе.
- Если техника разваливается — игрок видит, где сломался сокет.

### 5. Сохранение чертежа (Blueprint)
- Чертеж сохраняется в инвентарь.
- Можно продавать чертежи другим игрокам на бирже.

## 💡 Примеры сборок

### "Стрекоза" (Лёгкий разведывательный дрон)
- **Корпус:** Лёгкий каркас (Small)
- **Двигатели:** 4x Электрических пропеллера (Small)
- **Энергия:** 1x Малая батарея
- **Оружие:** 1x Лёгкий репетитор
- **Системы:** 1x Сканер
- *Итог:* Быстрый, слабый, идеален для разведки.

### "Титан" (Тяжёлый штурмовой танк)
- **Корпус:** Тяжёлая бронеплатформа (Large)
- **Движитель:** 2x Гусеничных модуля (Large)
- **Двигатель:** Дизельный V12 (Large)
- **Оружие:** 1x Рельсотрон (Large), 2x Репетиторы (Medium)
- **Защита:** Композитная броня + 1x APS
- *Итог:* Медленный, но пробивает любую технику с 1 выстрела.

## 🛠️ Технические детали для Unity (Реализация)

- **Хранение данных:** Все компоненты описаны в `ScriptableObject` (ItemData).
- **Сохранение техники:** Сериализация в JSON. Сохраняется только ID компонента, его локальная позиция, вращение и ID родительского сокета.
- **Физика:** `Unity Physics` (Havok) или встроенный PhysX. Для стыковки используется `FixedJoint` и `ConfigurableJoint`.
- **Оптимизация:** 
  - Объединение мешей (Mesh Combining) для статичных частей корпуса.
  - LOD-группы для собранной техники.
  - Отключение физики (`Rigidbody.isKinematic = true`) для техники дальше 500м от игрока.
