# ⚔️ Механика: Боевая система (Magnum Opus Combat System)

## 📖 Описание

**Боевая система** — это глубокая тактическая система с учётом баллистики, укрытий, типов урона и критических попаданий. Вдохновлена Escape from Tarkov, PUBG и Arma, но с упрощениями для MMO.

**Философия:** "Каждый выстрел на счету. Каждая позиция важна."

---

## ⚙️ Основные принципы

### 1. Реалистичная баллистика
Пули и снаряды имеют **физическую траекторию** (не hitscan).

**Параметры баллистики:**
- **Скорость пули** (Bullet Velocity) — 300-1200 м/с
- **Гравитация** (Bullet Drop) — пуля падает на расстоянии
- **Сопротивление воздуха** (Drag) — замедление пули
- **Рикошет** (Ricochet) — отскок от твёрдых поверхностей

**Пример:** Снайперская винтовка на 500м требует поправки на падение пули (~2 метра).

### 2. Система здоровья (Health System)
У игрока есть **общее здоровье** и **зоны поражения**.

**Зоны поражения:**
| Зона | Множитель урона | Эффект при попадании |
|------|-----------------|---------------------|
| **Голова** | x4.0 | Мгновенная смерть или оглушение |
| **Грудь** | x1.5 | Кровотечение, снижение точности |
| **Живот** | x1.2 | Сильное кровотечение, замедление |
| **Руки** | x1.0 | Снижение точности, дрожание |
| **Ноги** | x0.8 | Замедление, хромота |

**Механика кровотечения:**
- При попадании в конечности/живот — кровотечение
- Без лечения — потеря 5 HP/сек
- Лечение: бинты (останавливают), аптечки (лечат)

### 3. Система брони (Armor System)
Броня **поглощает урон** и имеет **классы защиты**.

**Классы брони:**
| Класс | Поглощение | Пробивается | Пример |
|-------|------------|-------------|--------|
| **Class 1** | 20% | Пистолеты, дробовики | Лёгкий жилет |
| **Class 2** | 40% | SMG, штурмовые винтовки | Тактический жилет |
| **Class 3** | 60% | Снайперские, пулемёты | Тяжёлый жилет |
| **Class 4** | 80% | Крупнокалиберные | Штурмовая броня |
| **Class 5** | 95% | Только бронебойные | Экзоскелет |

**Механика пробития:**
- Каждый патрон имеет **пенетрацию** (Penetration)
- Если пенетрация > класс брони — урон проходит
- Если пенетрация < класс брони — урон поглощается

### 4. Система укрытий (Cover System)
Игрок может **прятаться за объектами** для снижения урона.

**Типы укрытий:**
- **Полное укрытие** (Full Cover) — стена, камень, машина (100% защита)
- **Частичное укрытие** (Partial Cover) — куст, забор, ящик (50% защита)
- **Мягкое укрытие** (Soft Cover) — трава, песок (25% защита)

**Механика:**
- Игрок автоматически приседает за укрытием
- Можно выглядывать (Peek) для стрельбы
- Враг видит только часть тела (сложнее попасть)

## 💥 Типы урона (Damage Types)

### Кинетический (Kinetic)
- Пули, дробь, осколки
- Эффективен против: пехоты, лёгкой брони
- Неэффективен против: тяжёлой брони, техники

### Взрывной (Explosive)
- Гранаты, ракеты, мины
- Эффективен против: техники, укрытий, групп врагов
- Неэффективен против: рассредоточенных целей

### Термический (Thermal)
- Огнемёты, зажигательные гранаты
- Эффективен против: пехоты, деревянных построек
- Неэффективен против: металла, камня

### ЭМИ (EMP)
- Электромагнитные гранаты, ЭМИ-пушки
- Эффективен против: техники, электроники, щитов
- Неэффективен против: пехоты, простых укрытий

## 🎮 Геймплей боя

### 1. Подготовка
- Выбери оружие под ситуацию (ближний/дальний бой)
- Надень броню соответствующего класса
- Возьми аптечки, бинты, гранаты

### 2. Позиционирование
- Займи выгодную позицию (высота, укрытие)
- Контролируй линии огня (не стреляй по своим)
- Следи за флангами (не дай обойти)

### 3. Стрельба
- Целись в голову/грудь (максимальный урон)
- Контролируй отдачу (короткие очереди)
- Используй прицелы (коллиматор, оптика)

### 4. Лечение
- При ранении — укройся
- Останови кровотечение (бинты)
- Восстанови здоровье (аптечка)

### 5. Отступление
- Если перевес не на твоей стороне — отступай
- Используй дымовые гранаты для прикрытия
- Не беги по открытой местности

## ⚖️ Баланс

### Time-to-Kill (TTK)
- **Без брони:** 1-3 выстрела (зависит от оружия)
- **С бронёй Class 3:** 3-5 выстрелов
- **С бронёй Class 5:** 5-10 выстрелов

**Цель:** Быстрые бои (3-10 секунд), но с возможностью выжить при тактике.

### Экономика боя
- Патроны дорогие (10-100 койнов за штуку)
- Броня изнашивается (нужен ремонт)
- Лечение стоит денег (аптечки, бинты)

**Вывод:** Бой должен быть осмысленным, не спамить стрельбой.

## 🛠️ Технические детали для Unity

**Система:** Raycast + Projectile (гибридная)
- Ближнее оружие: Raycast (мгновенное попадание)
- Дальнее оружие: Projectile (физическая пуля)

**Скрипты:**
- `WeaponSystem.cs` — базовая логика оружия
- `BallisticsCalculator.cs` — расчёт траектории пули
- `DamageSystem.cs` — расчёт урона по зонам
- `ArmorSystem.cs` — поглощение урона бронёй

**Оптимизация:**
- Пул объектов для пуль (Object Pooling)
- Упрощённая физика для далёких пуль
- LOD для моделей оружия
